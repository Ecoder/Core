From ecoder v0.5 on, only the following browsers are supported:
* IE 9+
* FF 4+
* Safari 5.1+
* Chrome 13+
* Opera 12+
These limitations are set mainly by ECMAScript 5 support
	as seen on http://kangax.github.com/es5-compat-table/, accessed 30/10/2011

Only PHP 5.3 is supported, although we will try to be upwards compatible.
